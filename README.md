# Threema Dashboad Statistics

Threema  Dashboad Statistic is a web-based tool that allows you to upload, parse, and analyze your Threema chat exports. 

The app provides detailed statistics about your messages, top users, most common words, and an interactive dashboard for easy navigation.

[Live Demo: ](https://spawntec.com/threemaboard/)

## Features

* Upload Multiple Threema chat files.
* Automatically parses messages with date, time, user, and content.
* Display total messages, sent and received messages, and unique users.
* Identify the most active users and most commonly used words.
* Filter messages by user, text, or date.
* Paginated message display with adjustable messages per page.
* Drag-and-drop dashboard cards to customize your view.
* Smooth scroll-to-top button for easy navigation.
* 100% Privat works offline in Browser only.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/carlostkd/threema-dashboard.git
cd threema-dashboard
```

Serve the file

```
python -m http.server 8000

php -S 127.0.0.1:8000 -t .

```

2. Open in your browser. No backend required, fully client-side.
